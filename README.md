# Hotel Booking Cancellation Prediction - MLOps

This repository contains the production deployment and MLOps implementation for the **Hotel Booking Cancellation Prediction** machine learning project developed as part of the MSc Data Science dissertation.

The project demonstrates an end-to-end MLOps workflow by deploying a trained **CatBoost** model as a RESTful API using **FastAPI** and **Docker**, while integrating **MLflow** on **Databricks** for experiment tracking, model versioning, and monitoring.

## Technologies

- Python
- CatBoost
- FastAPI
- Docker
- MLflow
- Databricks
- GitHub Actions
- AWS

## Project Objectives

- Deploy the trained machine learning model as a REST API
- Containerize the application using Docker
- Track experiments and model versions using MLflow
- Design a production-ready MLOps pipeline
- Demonstrate cloud deployment, monitoring, and model lifecycle management

> **Note:** This repository focuses on model deployment and MLOps. Model development, exploratory data analysis, feature engineering, and model evaluation are maintained in a separate machine learning repository.
